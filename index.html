Multiple Disease Detection using Machine Learning

This project demonstrates the use of machine learning models to predict multiple diseases including Diabetes, Heart Disease, and Parkinson's Disease.

Features

  • Diabetes Prediction based on various health parameters
  • Heart Disease Prediction using clinical data
  • Parkinson's Disease Prediction from voice measurements

How to Use

To run the application locally:

  1. Install required packages: pip install streamlit pickle numpy streamlit-option-menu
  2. Run the app: streamlit run multiplediseaseprediction.py

Files

  • multiplediseaseprediction.py - Main Streamlit application
  • diabetes_model.sav, heart_disease_model.sav, parkinsons_model.sav - Trained ML models
  • diabetes.csv, heart.csv, parkinsons.csv - Datasets used for training
  • Jupyter notebooks for model training and analysis

Note

This is a demonstration project. For actual medical diagnosis, please consult healthcare professionals.
